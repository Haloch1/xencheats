XENCHEATS — category (game) art
================================
Drop images here to give each game card a background photo and a logo.
Missing files fall back to the colored gradient + initials automatically.

Per game, two optional files:
  <slug>.jpg        background photo (fills the card, wide/landscape works best)
  <slug>-logo.png   logo (transparent PNG, sits centered on the card)

Game slugs:
  r6        -> r6.jpg / r6-logo.png            (Rainbow Six Siege)
  fortnite  -> fortnite.jpg / fortnite-logo.png
  apex      -> apex.jpg / apex-logo.png        (Apex Legends)
  tarkov    -> tarkov.jpg / tarkov-logo.png    (Escape From Tarkov)
  rust      -> rust.jpg / rust-logo.png
  spoofer   -> spoofer.jpg / spoofer-logo.png
  accounts  -> accounts.jpg / accounts-logo.png

Background may also be .png (<slug>.png) if you prefer.
